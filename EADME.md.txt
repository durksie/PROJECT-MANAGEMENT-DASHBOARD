# Project Management Dashboard

A modern, responsive project management dashboard built with React and Tailwind CSS. Manage projects, team members, events, and collaborate with your team all in one place.

## Features

- **Home Dashboard** - Overview of all projects, events, and team statistics
- **Add Project** - Create new projects with title, dates, description, tech stack, and team member assignment
- **Projects Listing** - View all projects with detailed information including status, dates, tech stack, and team members
- **Events** - Track upcoming events associated with your projects
- **Community** - View all team members and their information
- **Responsive Design** - Fully responsive on mobile, tablet, and desktop devices
- **Interactive Member Selection** - Assign existing team members or add custom member names to projects
- **Search & Filter** - Find projects and information quickly

## Tech Stack

- **React** - UI library
- **Next.js** - React framework
- **Tailwind CSS** - Utility-first CSS framework
- **TypeScript** - Type-safe JavaScript
- **shadcn/ui** - Component library
